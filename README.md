# Claude Code Usage Bar

macOS menu bar app that shows your Claude Code API usage limits at a glance.

Displays 5-hour session, 7-day week, and 7-day Sonnet utilization with reset times.

## Install

```bash
git clone https://github.com/meule/ai_usage_bar.git
cd ai_usage_bar
bash build.sh
open ClaudeUsageBar.app
```

Requires macOS 13+ and an active [Claude Code](https://docs.anthropic.com/en/docs/claude-code) OAuth session.
